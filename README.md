# Flog


### 우물 밖이 궁금한 당신의 여행 가이드북
여행을 끝내고 집에 돌아온 뒤에는 완성된 나만의 가이드북을 다른 사람과 공유할 수 있도록 하여

사용자들 간에 양질의 여행 정보를 나누고, ‘**나의 여행**’이 '**우리의 여행**'으로 확장되게 합니다.
